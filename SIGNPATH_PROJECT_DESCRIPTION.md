## 짧은 설명

`ebook_viewer` is an open source Windows desktop application built with Electron. It is used as a textbook viewer and is distributed as a signed Windows executable.

## 자세한 설명

`ebook_viewer` is a public open source Electron application for Windows.

The project provides a desktop viewer application with the following characteristics:

- built with Electron
- packaged for Windows distribution
- distributed as `ebook_viewer.exe`
- released from a public source repository with versioned tags

The source code, build configuration, and release documentation are published in the public repository so that distributed binaries can be traced back to the tagged source.
